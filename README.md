# BVG tracker

A Python tool for collecting real-time BVG Radar data and saving historical 
trajectories of Bus 142 (Berlin, Germany) into CSV format.

## Features
- Collect real-time BVG Radar data (by line & bounding box)
- Save trajectory points to CSV (timestamp, lat, lon, heading, speed)
- Configurable interval, line number, bounding box
- Extendable for visualization and analysis
